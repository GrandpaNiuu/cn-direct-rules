# Adopt hourly ShareAlike BGP data

To close independently observed IPv4 routing gaps, the canonical aggregate incorporates `misakaio/chnroutes2`, an hourly BGP source under CC BY-SA 4.0. This adds useful routed coverage but means generated artifacts containing the canonical IPv4 aggregate must preserve Misaka Network attribution and ShareAlike terms; repository code remains MIT-licensed, and APNIC registered-allocation data remains a separate audit because registration country is not current geolocation.
